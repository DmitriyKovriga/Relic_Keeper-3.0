## Как подготовить свой шрифт
1. Положи `.ttf` или `.otf` в проект, например:
   - `Assets/Fonts/MyFont/MyFont-Regular.ttf`
2. Для UI Toolkit:
   - используй этот `Font` напрямую (поле `UI Toolkit Font` в профиле).
3. Для TextMeshPro:
   - открой `Window > TextMeshPro > Font Asset Creator`
   - выбери свой `.ttf/.otf` в `Source Font File`
   - `Atlas Population Mode`: `Dynamic`
   - нажми `Generate Font Atlas`, затем `Save`
   - сохрани, например: `Assets/Fonts/MyFont/TMP/MyFont SDF.asset`
4. Назначь оба шрифта в `UIFontProfile`.

## Как использовать окно
1. Открой `Tools/UI/Font Manager`
2. Выбери `UIFontProfile` (или нажми `Создать`)
3. Назначь:
   - `UI Toolkit Font`
   - `TMP Font Asset`
4. Проверь блок **Превью до применения**:
   - EN строка
   - RU строка
5. Проверь блок **Проверка глифов**:
   - если есть ошибки, в шрифте нет нужных символов
6. Нажми `Dry Run` (без изменений)
7. Нажми `Применить к проекту`

## Важно по безопасности
- Перед применением сохраняй сцены.
- Применение меняет много ассетов (сцены/префабы) — это нормально.
- Всегда делай через git-коммит, чтобы можно было откатить.
- После смены шрифта проверь UI в игре: у разных шрифтов разная метрика, возможны сдвиги текста.
