# Спецификация арта склада (Stash) — актуальный контракт

## Базовый контекст
- Экран: `480x270`
- Правое окно инвентаря: `275x270`
- Левое окно склада (под арт): `200x270`
- Зазор между окнами: `5 px`

Источник: `Assets/UI/BasicUIScripts/InventoryCompanionLayout.cs`

## Что нужно рисовать (по текущей реализации)
1. `stash_panel_full` — основной фон склада с интегрированной рамкой сетки: `200x270`
2. `stash_tab_bg` — подложка обычной вкладки/кнопки `+`: `20x14`
3. `stash_tab_close_bg` — подложка кнопки `x`: `12x12`

Сейчас отдельная рамка сетки не используется: рамка встроена в `stash_panel_full`.

## Геометрия внутри `stash_panel_full` (координаты в пикселях)
Координаты считаются от левого верхнего угла окна склада `200x270`.

- Внешний прямоугольник окна: `x=0, y=0, w=200, h=270`
- Отступы контента (padding):
  - слева: `8`
  - справа: `8`
  - сверху: `8`
  - снизу: `20`
- Контентная область (после отступов): `x=8, y=8, w=184, h=242`
- Зона вкладок (контейнер): `x=8, y=12, w=184, h=18`
- Зона сетки слотов (контейнер): `x=8, y=30, w=180, h=220`
- Сетка: `9` колонок x `11` рядов
- Размер слота: `20x20`

Важно для художника:
- Линии рамки/сетки в `stash_panel_full` должны попадать точно в сетку `x=8..188`, `y=30..250`.
- Если у рамки есть декоративные выступы, они должны оставаться в пределах `200x270`.

## Поведение вкладок
- `stash_tab_bg` (`20x14`) используется для:
  - вкладок `1,2,3...`
  - кнопки `+`
- `stash_tab_close_bg` (`12x12`) используется для кнопки `x` у активной вкладки.

Текст на вкладках рисуется движком, не в спрайте.

## Технический формат импорта (Unity)
- Texture Type: `Sprite (2D and UI)`
- Sprite Mode: `Multiple` (если в одном атласе) или `Single` (если раздельные файлы)
- Filter Mode: `Point (no filter)`
- Compression: без потерь для пиксель-арта
